Volunteer computing

Defined as a set of tools that allow citizens to donate cycles to scientific experiments by running an application.

Volunteers are:

... anonymous.

... not accountable.

... trusting the application provider.

A JavaScript Based Solution?

  • Experiments are embedded in a web page.
  • Browsers are ubiquitous
  • JavaScript also runs outside the browser in nodejs, and compiled from other languages
  • HTML5 is catching up.
