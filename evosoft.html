We Need more Computing Power

Volunteer computing

Defined as a set of tools that allow citizens to donate cycles to scientific experiments by running an application.

A JavaScript Based Solution?

  • Experiments are embedded in a web page.
  • Browsers are ubiquitous: Desktops, Tablets, TVs, Smartphones.
  • JavaScript also runs compiled in servers.
  • HTML5 is caching up: Web Workers, Canvas, Storage, Sockets.

In a slide

  • We are presenting a JavaScript based framework for running distributed volunteer-based evolutionary algorithm experiments.
  • We measured the speed of JavaScript in evolutionary algorithms.
  • This can serve as a baseline for comparison with other distributed computing experiments.
